# Compatibility Matrix

## Version 3.0.0

| Spring Data JPA | Spring Framework | Spring Boot | Java Version | Jakarta Persistence |
|-----------------|------------------|-------------|--------------|---------------------|
| 4.0.6+          | 7.0.8+           | 4.0.6+      | 25           | 3.2.0+              |

## Historical Note

The `3.0.0` line intentionally supports only the Spring `7` / Spring Boot `4` generation.

If you need support for older stacks such as Spring Framework `6.x`, Spring Boot `3.x`, or Java versions below `25`, use the `2.x` release line.
